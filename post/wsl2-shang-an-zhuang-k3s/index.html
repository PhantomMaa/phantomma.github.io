WSL2 дёҠе®үиЈ… K3S

В· 2023-08-24 В·

еңЁwsl2 ubuntu-22.04пјҢдҪҝз”Ёk3sе®ҳзҪ‘зҡ„дёҖй”®е®үиЈ…е‘Ҫд»ӨпјҢжҳҜе®үиЈ…дёҚжҲҗеҠҹзҡ„гҖӮ

curl -sfL https://get.k3s.io | sh - 
# Check for Ready node, takes ~30 seconds 
sudo k3s kubectl get node

еҺҹеӣ пјҢk3sзҡ„иҝҗиЎҢиҝҳдҫқиө–systemdгҖӮеңЁжү§иЎҢдёҠйқўе‘Ҫд»Өд№ӢеүҚпјҢ

е…ҲиҰҒеңЁwsl2 дёҠејҖеҗҜsystemd

еңЁwsl2 дёҠж–°е»әж–Үд»¶пјҡ/etc/wsl.confпјҢж·»еҠ д»ҘдёӢеҶ…е®№пјҡ

[boot]
systemd=true

йҮҚеҗҜwsl2

еңЁwindows11 дёҠжү§иЎҢwsl --shutdownпјҢз„¶еҗҺејҖеҗҜж–°зҡ„wslпјҢwslгҖӮ
еңЁubuntuдёҠжү§иЎҢе‘Ҫд»ӨпјҢйӘҢиҜҒsystemd зҡ„ејҖеҗҜж•ҲжһңгҖӮ
systemctl list-unit-files --type=service

е®үиЈ… docker

https://docs.docker.com/engine/install/ubuntu/

docker е…Қ sudo

https://www.docker.org.cn/book/install/run-docker-without-sudo-30.html

йҮҚж–°жү§иЎҢk3s зҡ„дёҖй”®е®үиЈ…и„ҡжң¬

еҲҷдјҡзңӢеҲ°е®үиЈ…жҲҗеҠҹзҡ„иҫ“еҮәдәҶпјҡ

# дҪҝз”Ё docker иҝҗиЎҢж—¶
вһң  ~ curl -sfL https://get.k3s.io | sh -s - --docker
# Check for Ready node, takes ~30 seconds
sudo k3s kubectl get node
....
[INFO]  systemd: Starting k3s
NAME            STATUS   ROLES                  AGE   VERSION
nb-hz20314960   Ready    control-plane,master   14m   v1.28.4+k3s2

е…Қsudo жү§иЎҢkubectl

жӣҙж”№зҡ„ж–Үд»¶жқғйҷҗпјҡ
sudo chmod 644 /etc/rancher/k3s/k3s.yaml
йҮҚеҗҜk3s
sudo systemctl restart k3s
з„¶еҗҺжү§иЎҢ kubectl get node еҲҷдёҚеҶҚйңҖиҰҒrootжқғйҷҗгҖӮ

еҫ—еҲ°k3sзҡ„kubeconfig

жҺҘдёӢжқҘеҸҜд»ҘеҺ»иҝҷдёӘи·Ҝеҫ„еӨҚеҲ¶ kubeconfigпјҢж·»еҠ еҲ°дҪ ж“ҚдҪңз”өи„‘зҡ„kubeconfigйҮҢпјҢеҚіеҸҜи·ҹWSL2 дёӢзҡ„k3sиҝӣиЎҢдәӨдә’дәҶгҖӮ
/etc/rancher/k3s/k3s.yaml
